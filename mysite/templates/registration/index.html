This is the index page for the GPS team challenge data analysis web site.

UNDER DEVELOPMENT all DATA IS TEMPORARY

The purpose of this web site is to allow a user to read the data posted into the main GPS Team Challenge web site at www.gpsteamchallenge.com.au into this site. Then add to it extra information such as the location of each posting and the equipment used. This will allow the use to then get information like personal best results at each location or with each combination of equipment. Duplicating the data here allows the analysis to be done without any risk or extra load on the "real" gpsteamchallenge site.
